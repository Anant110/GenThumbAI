16. “In React Router, <Link> is used for declarative navigation, meaning we use it in JSX to let users navigate by clicking links, similar to an anchor tag but without reloading the page.
On the other hand, useNavigate is used for programmatic navigation, where navigation happens through JavaScript logic, like redirecting a user after login or based on some condition.
So basically, <Link> is used for UI-based navigation, while useNavigate is used for logic-based navigation.”

### 📌 AspectRatioSelector (Notes)
**Purpose:**
A reusable React component used to let users select an aspect ratio (like 16:9, 1:1, 9:16) with visual feedback.
---
### 🔹 Props
* `value: AspectRatio` → currently selected ratio
* `onChange: (ratio: AspectRatio) => void` → function to update selected ratio
---
### 🔹 Data Used
* `aspectRatios` → list of available ratios
* `AspectRatio` → TypeScript type for ratio values
---
### 🔹 Core Logic
1. Loop through all aspect ratios using `.map()`
2. For each ratio:
   * Check if it is selected:
     ```
     selected = value === ratio
     ```
   * Render a button
3. On button click:
   * Call `onChange(ratio)` to update parent state
---
### 🔹 Icon Mapping
* Uses `iconMap` to assign icons for each ratio:
  * 16:9 → horizontal icon
  * 1:1 → square icon
  * 9:16 → vertical icon
---
### 🔹 Styling Logic
* If `selected` → apply highlight style (`bg-white/10`)
* Else → apply hover effect (`hover:bg-white/6`)
---
### 🔹 Key Concept
* This is a **controlled component**
  * Parent controls `value`
  * Child triggers updates using `onChange`
---
### 🔹 Flow
1. Parent sends `value`
2. Component renders buttons
3. Selected button is highlighted
4. User clicks a button
5. `onChange` is called
6. Parent updates state
7. Component re-renders
---
### 🔹 One-line Summary
Displays selectable aspect ratio options with icons and highlights the currently selected one.
